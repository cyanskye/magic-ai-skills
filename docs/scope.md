# 收录范围

本仓库只收录自己创建并稳定使用的 Magic AI 工作流 Skills。

## 当前收录

- `getnote`
- `magic-recorder`
- `magic-kb-compiler`

## 不收录

- 第三方 Skills
- 系统内置 Skills
- 插件缓存 Skills
- 只是在本机安装过、但不是自己创建维护的 Skills
- 历史版本
- 真实密钥、cookie、token、私密笔记内容

## 判定规则

一个 Skill 进入本仓库，需要同时满足：

1. 是自己创建或长期维护的工作流。
2. 当前版本稳定可用。
3. 与 Magic AI 工作流直接相关。
4. 依赖关系清楚。
5. 不包含真实凭据或私密内容。
